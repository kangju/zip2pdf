# ZIP2PDF
## 概要(What is ZIP2PDF?)
ZIPファイルで圧縮されたJPGファイル・PNGファイルをPDFに変換します。  
ZIPファイルと同じディレクトリにZIPファイルと同じ名前で保存されます。

## 実行環境(environment)
python 3.9

## 依存関係(requirement)  
[pipenv](https://github.com/pypa/pipenv)

## 使い方(How to use)
``` shell
./zip2pdf.sh <zip/file/path>
```
